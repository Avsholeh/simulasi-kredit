# Simulasi Kredit
Aplikasi Simulasi Kredit dengan perhitungan suku bunga flat, efektif dan anuitas.

## Dependensi
Aplikasi ini akan berjalan diatas sekumpulan teknologi & pustaka (library) dibawah ini:

- PHP v.7.2.19
- Bootstrap v.4.4.1
- jQuery v.3.4.1
- NumeralJS v.2.0.6
